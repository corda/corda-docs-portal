      • KeyRotationStatus

        KeyRotationStatus(String requestId, KeyType managedKey, String oldParentKeyAlias, String newParentKeyAlias, Integer oldGeneration, String tenantId, Integer newGeneration, Integer processedCount, Integer expectedTotal, Instant createdTimestamp, Instant lastUpdateTimestamp)
        All-args constructor.
        Parameters:
        requestId - GUID allocated by client and passed in StartRotation message.
        managedKey - Type of the key to be rotated.
        oldParentKeyAlias - The wrapping key alias that should no longer be used, and all its protected content re-wrapped with a new key.
        newParentKeyAlias - The wrapping key alias that should be used for material currently wrapped with old key.
        oldGeneration - Specifies the old generation number we should move away from.
        tenantId - Only specified when targeting managed keys.
        newGeneration - Set to the new generation number when rotating a single managed key, otherwise null.
        processedCount - Number of wrapped keys rotated so far.
        expectedTotal - Number of wrapped keys in total to re-wrap as part of this operation.
        createdTimestamp - The date and time the key rotation request was created.
        lastUpdateTimestamp - The date and time when the last individual key rotation operation has finished.
