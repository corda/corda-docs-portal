Specification

abstract class Specification<VALUE> : Schema, Parser<VALUE>

A Configuration.Schema that is also able to parse a raw Config object into a VALUE. It is an abstract class to allow extension with delegated properties e.g., object Settings: Specification() { val address by string().optional("localhost:8080") }.
