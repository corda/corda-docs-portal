open fun toToken(context: SerializeAsTokenContext): SingletonSerializationToken

Creates the singleton's token and returns it.

Prefer providing an override of tokenName to alter the SingletonSerializationToken output from this method. The context should not be used here, as registration of the SingletonSerializationToken is done by Corda.

Parameters

context - The SerializeAsTokenContext involved in registering the token.

Return
A SingletonSerializationToken representing the singleton's token.
