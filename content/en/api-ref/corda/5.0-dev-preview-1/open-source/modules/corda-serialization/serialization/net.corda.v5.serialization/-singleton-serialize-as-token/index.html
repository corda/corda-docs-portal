SingletonSerializeAsToken

interface SingletonSerializeAsToken : SerializeAsToken

This interface should be implemented by classes that want to substitute a singleton token representation of themselves if they are serialized because they have a lot of internal state that does not serialize (well).

This interface should only be used on singleton classes, meaning that only one should exist during runtime.
