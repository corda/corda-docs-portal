abstract fun load(stateRef: StateRef): StateAndRef<ContractState>

Given a StateRef loads the referenced transaction and looks up the specified output ContractState.

WARNING Do not use this method unless you really only want a single state - any batch loading should go through loadSet as repeatedly calling loadStateRef can lead to repeat deserialization work and severe performance degradation.

Parameters

stateRef - The StateRef to retrieve a StateAndRef with.

Exceptions

TransactionResolutionException - If the stateRef points to a non-existent transaction.

Return
The retrieved StateAndRef.
