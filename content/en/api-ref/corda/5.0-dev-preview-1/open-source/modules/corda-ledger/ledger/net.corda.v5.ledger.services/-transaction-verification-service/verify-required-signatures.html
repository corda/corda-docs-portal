verifyRequiredSignatures

abstract fun verifyRequiredSignatures(transaction: TransactionWithSignatures): Unit

Verifies the signatures on the input transaction and throws if any are missing. In this context, "verifying" means checking they are valid signatures and that their public keys are in the TransactionWithSignatures.requiredSigningKeys set.

Parameters

transaction - The TransactionWithSignatures that will have its signatures verified.

Exceptions

SignatureException - If any signatures are invalid or unrecognised.

SignaturesMissingException - If any signatures should have been present but were not.

InvalidKeyException - If the key on a signature is invalid.
