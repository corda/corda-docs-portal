abstract fun sign(builder: TransactionBuilder, publicKeys: Collection<PublicKey>): SignedTransaction

Constructs an initial partially signed transaction from a TransactionBuilder using a set of keys all held in this node.

Parameters

builder - The TransactionBuilder to seal with the node's signature. Any existing signatures on the builder will be preserved.

publicKeys - A list of PublicKeys used to lookup the matching PrivateKey and sign.

Exceptions

IllegalArgumentException - If any keys are unavailable locally.

Return
