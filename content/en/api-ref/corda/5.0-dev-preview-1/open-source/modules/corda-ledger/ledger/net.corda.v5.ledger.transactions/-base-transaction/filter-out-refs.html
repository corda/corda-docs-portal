filterOutRefs

open fun <T : ContractState> filterOutRefs(clazz: Class<T>, predicate: Predicate<T>): List<StateAndRef<T>>

Helper to simplify filtering output StateAndRef items according to a Predicate.

Parameters

clazz - The class type used for filtering via an Class.isInstance check. Clazz must be an extension of ContractState.

predicate - A filtering function taking a state of type T and returning true if it should be included in the list. The class filtering is applied before the predicate.

Return
the possibly empty list of output StateAndRef states matching the predicate and clazz restrictions.
