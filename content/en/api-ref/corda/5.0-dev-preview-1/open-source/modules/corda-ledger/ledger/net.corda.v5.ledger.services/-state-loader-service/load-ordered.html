loadOrdered

abstract fun loadOrdered(stateRefs: List<StateRef>): List<StateAndRef<ContractState>>

Given a List of StateRef's loads each referenced transaction and looks up the specified output ContractStates.

The returned list will be in the same order as the input stateRefs list.

Exceptions

TransactionResolutionException - If any of the stateRefs point to a non-existent transaction.

Return
The retrieved set of StateAndRefs.
