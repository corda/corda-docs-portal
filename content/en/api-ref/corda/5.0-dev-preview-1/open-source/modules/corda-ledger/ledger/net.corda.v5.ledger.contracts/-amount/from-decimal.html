fromDecimal

@JvmStatic fun <T : Any> fromDecimal(displayQuantity: BigDecimal, token: T, rounding: RoundingMode): Amount<T>

Build an Amount from a decimal representation. For example, with an input of "12.34 GBP", returns an amount with a quantity of "1234" tokens. The function getDisplayTokenSize is used to determine the conversion scaling, for example bonds might be in nominal amounts of 100, currencies in 0.01 penny units.

Parameters

displayQuantity - The BigDecimal to generate the Amount's value from.

token - The token of the Amount.

rounding - The rounding mode placed on the value.

T - The type of the token.

Exceptions

ArithmeticException - if the intermediate calculations cannot be converted to an unsigned 63-bit token amount.

An Amount
