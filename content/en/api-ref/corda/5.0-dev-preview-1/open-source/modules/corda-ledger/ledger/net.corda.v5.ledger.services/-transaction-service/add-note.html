addNote

abstract fun addNote(txId: SecureHash, noteText: String): Unit

Adds a note to an existing LedgerTransaction given by its unique SecureHash id. Multiple notes may be attached to the same LedgerTransaction. These are additively and immutably persisted within the node local vault database in a single textual field using a semi-colon separator.

Parameters

txId - The transaction to add the note to.

noteText - The text to add.
