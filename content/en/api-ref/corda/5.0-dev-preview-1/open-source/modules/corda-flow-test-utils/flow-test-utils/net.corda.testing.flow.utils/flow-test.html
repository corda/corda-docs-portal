fun <T : Flow<*>> flowTest(flowClass: Class<T>?, testCode: FlowTest<T>): Unit

Instantiates a new FlowMockHelper and executes the given test code block with that instance passed in.

Example usage in Java:

flowTest(ReceiveTransactionFlow.class, fmh -> {
    fmh.createFlow(it -> new ReceiveTransactionFlow(fmh.getOtherSideSession()));

    SignedTransaction result = null;
    try {
        result = fmh.getFlow().call();
    } catch (Exception e) {
        fail("ReceiveTransactionFlow threw an exception.", e);
    }
    assertEquals(fmh.getMockSignedTransaction(), result);

    ReceiveTransactionFlow flow = fmh.getFlow();
    verify(flow.flowEngine).subFlow(Mockito.any(ResolveTransactionsFlow.class));
});
