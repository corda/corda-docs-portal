abstract fun <U, V : U> overrideDefaultInjectableMock(interfaceClass: Class<U>, implementation: V): Unit

Override the default mocks created by the FlowMockHelper with a custom mock for classes which are injectable into flows. This should be called before creating a flow, i.e. before calling createFlow.

Parameters

interfaceClass - interface which is injectable into flows.

implementation - an implementation of interfaceClass which will be injected into the flow.
