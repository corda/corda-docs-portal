sleep

@Suspendable abstract fun sleep(duration: Duration): Unit

Suspends the flow and only wakes it up after at least duration time has passed.

Warning: long sleeps and in general long running flows are highly discouraged, as there is currently no support for flow migration!

Exceptions

FlowException - if attempted to sleep for longer than 5 minutes.
