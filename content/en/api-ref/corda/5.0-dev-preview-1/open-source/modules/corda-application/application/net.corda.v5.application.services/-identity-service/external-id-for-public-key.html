externalIdForPublicKey

abstract fun externalIdForPublicKey(publicKey: PublicKey): UUID?

Returns the external ID (as an UUID) related to the passed in PublicKey. Providing a PublicKey that is unknown by the node or is not mapped to an external ID will return null. The method looks up keys generated by the node as well as keys generated on other nodes and registered with the IdentityService.

Parameters

publicKey - The PublicKey used to perform the lookup to external Id.
