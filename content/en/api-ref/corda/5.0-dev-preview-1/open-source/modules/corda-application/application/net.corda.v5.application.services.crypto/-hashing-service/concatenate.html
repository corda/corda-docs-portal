concatenate

abstract fun concatenate(first: SecureHash, second: SecureHash): SecureHash

Appends second digest to first, and then computes the digest of the result. Both digests need to be of the same digest algorithm. The result digest is of the same digest algorithm as well.

Parameters

first - The first digest in the concatenation.

second - The second digest in the concatenation.
