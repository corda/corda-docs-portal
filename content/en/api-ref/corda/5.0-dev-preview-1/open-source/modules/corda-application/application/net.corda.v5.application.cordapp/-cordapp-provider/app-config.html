appConfig

abstract val appConfig: CordappConfig

Exposes the current CorDapp configuration.

The calling application is found via stack walking and finding the first class on the stack that matches any class contained within the automatically resolved Cordapps loaded by the net.corda.nodeapi.internal.cordapp.CordappLoader

Exceptions

IllegalStateException - When called from a non-app context.

Return
The configuration for the current CorDapp.
