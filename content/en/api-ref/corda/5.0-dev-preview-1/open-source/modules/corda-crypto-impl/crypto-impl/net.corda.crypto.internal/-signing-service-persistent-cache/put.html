abstract fun put(key: Any, entity: SigningPersistentKey): SigningPersistentKey

Persist the specified value and associates it with the specified key in this map (in that order). If the underlying storage previously contained a value for the key, the behaviour is unpredictable and most likely will throw an error from the underlying storage.
