KeyManagementBackend

interface KeyManagementBackend

The KeyManagementBackend provides operation support for KeyManagementService and is responsible for storing and using private keys to sign things. An implementation of this may, for example, call out to a hardware security module that enforces various auditing and frequency-of-use requirements.
