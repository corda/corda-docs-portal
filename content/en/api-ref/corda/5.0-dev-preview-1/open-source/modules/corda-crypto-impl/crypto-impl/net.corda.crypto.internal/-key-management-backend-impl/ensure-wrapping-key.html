ensureWrappingKey

open fun ensureWrappingKey(): Unit

Ensures that the wrapping key is generated if required by the underlying CryptoService implementation. The alias under which the created wrapping key is created would have to be provided in the configuration. The method would be called only during the bootstrapping process.

Exceptions

CryptoServiceException - for general cryptographic exceptions.
