createCertificate

fun SigningService.createCertificate(certificateType: CertificateType, issuerCertificate: X509Certificate, issuerPublicKey: PublicKey, subject: X500Principal, subjectPublicKey: PublicKey, validityWindow: Pair<Duration, Duration> = DEFAULT_VALIDITY_WINDOW, nameConstraints: NameConstraints? = null, crlDistPoint: String? = null, crlIssuer: X500Name? = null): X509Certificate

Create a X509 v3 certificate using the given issuer certificate and key pair.

Parameters

certificateType - type of the certificate.

issuerCertificate - The Public certificate of the root CA above this used to sign it.

issuerPublicKey - The KeyPair of the root CA above this used to sign it.

subject - subject of the generated certificate.

subjectPublicKey - subject's public key.

validityWindow - The certificate's validity window. Default to DEFAULT_VALIDITY_WINDOW if not provided.

nameConstraints - any name constraints to impose on certificates signed by the generated certificate.

crlDistPoint - CRL distribution point.

crlIssuer - X500Name of the CRL issuer.

Return
A data class is returned containing the new intermediate CA Cert and its KeyPair for signing downstream certificates. Note the generated certificate tree is capped at max depth of 1 below this to be in line with commercially available certificates.
