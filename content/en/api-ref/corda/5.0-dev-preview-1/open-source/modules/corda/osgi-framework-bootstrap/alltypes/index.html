All Types

net.corda.osgi.framework.OSGiFrameworkException

Extends the BundleException class keeping a reference to the location of the bundle at the origin of the exception.

net.corda.osgi.framework.OSGiFrameworkMain

The OSGiFrameworkMain class provides the static methods to bootstrap the Apache FelixOSGi framework accessible in the class-path.

net.corda.osgi.framework.OSGiFrameworkWrap

Wrap the OSGi framework keeping a map of the installed bundles and providing methods to manage bundles. Used by Corda to bootstrap an OSGi framework, install the required bundles and the Corda node bundle before to bootstrap the Corda node. Thread safe.
