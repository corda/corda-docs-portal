activate

@Synchronized fun activate(): OSGiFrameworkWrap

Activate the installed bundles not active yet. Activate only the bundles previously installed by this. Synchronized.

Exceptions

OSGiFrameworkException - If any bundle can't be activated.

IllegalStateException - if any bundle was uninstalled by a concurrent process accessing the OSGi framework when the bundle has to be activated by this method.

SecurityException - If the caller does not have the appropriate org.osgi.framework.AdminPermission.EXECUTE permission for the wrapped OSGi framework, and the Java Runtime Environment supports permissions.

Return
This.
