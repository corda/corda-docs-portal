install

@Synchronized fun install(file: File): OSGiFrameworkWrap

Install the bundle available as 'file' in the OSGi framework wrapped by this. OSGi fragments are not subject to activation. Recursive: if a directory is found, it's content is scanned and jars installed. Synchronized.

Parameters

file - Bundle to install in the OSGi framework. Files having extension different from jarExtension are ignored.

Exceptions

IllegalStateException - If the org.osgi.framework.BundleContext of the OSGi framework, wrapped by this, is invalid when the bundle is installed.

SecurityException - If a security manager exists and denies read access to the file.

Return
This.
