bootstrap

@Synchronized fun bootstrap(args: Array<String>): OSGiFrameworkMain

It bootstraps the OSGI framework exposing from the JVM to the OSGi framework the packages listed in the OSGiFrameworkMain.SYSTEM_PACKAGES_EXTRA_RESOURCE resource, then it loads and activates the bundles listed at OSGiFrameworkMain.SYSTEM_BUNDLES_RESOURCE. If the bundles can't be activated or any exception is thrown from the wrapper OSGi framework, this method log the exceptions and shutdown the framework. The method creates a temporary storage directory for bundles, the directory has OSGiFrameworkMain.OSGI_FRAMEWORK_STORAGE_PREFIX prefix and it is removed automatically when the JVM terminates.

Parameters

args - Passed to the OSGiFrameworkWrap.setArguments to be notified to the Corda node inside the OSGI framework.

Exceptions

ClassNotFoundException - If the OSGI FrameworkFactory can't be loaded from this classpath/jar.

IllegalArgumentException - If OSGiFrameworkMain.OSGI_FRAMEWORK_STORAGE_PREFIX cannot be used to generate a candidate directory name.

IOException - If the resources listing the packages to expose to the wrapped OSGi framework, and the bundles to load in it, those can't be accessed. If an I/O error occurs or the temporary-file directory can't me created.

SecurityException - If a security manager is installed and it check this process can't create a temporary directory or delete it when the JVM terminates.

Return
this.
