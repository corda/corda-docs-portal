start

@Synchronized fun start(): OSGiFrameworkWrap

Start the wrapped OSGi framework. This method add a threadas shutdown hook a thread executed when the virtual machine begins its shutdown sequence: the thread call the stop method to shutdown the OSGi framework, the OSGi framework calls the org.osgi.framework.BundleActivator.stop methods where the code to stop each bundle - if needed - should run.

Exceptions

IllegalStateException - If the bundle of the wrapped OSGi framework has been uninstalled or it tries to change its own state. If the virtual machine is shutting down when start() is called. If the shutdown hook has been already been registered, since this method registers the hoot to execute stop when the process shutdowns.

OSGiFrameworkException - If the wrapped OSGi framework could not be started.

SecurityException - If the caller does not have the appropriate AdminPermission.EXECUTE, and the Java Runtime Environment supports permissions.
