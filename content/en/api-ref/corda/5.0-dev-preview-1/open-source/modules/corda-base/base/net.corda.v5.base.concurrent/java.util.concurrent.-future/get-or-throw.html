getOrThrow

fun <V> Future<V>.getOrThrow(timeout: Duration?): V

Waits if necessary for the computation to complete, and then retrieves its result.

This function is the same as Future.get except that it throws the underlying exception by unwrapping the ExecutionException.

Parameters

timeout - The maximum duration to wait until the future completes.

Exceptions

CancellationException - If the computation was cancelled.

InterruptedException - If the current thread was interrupted while waiting.

TimeoutException - If the wait timed out.

Return
The computed result.
