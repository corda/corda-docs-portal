createWrappingKey

abstract fun createWrappingKey(masterKeyAlias: String, failIfExists: Boolean): Unit

Generates a new key to be used as a wrapping key. Some of the implementations may not have the notion of the wrapping key in such cases the implementation should do nothing. The method would be called only during the bootstrapping process.

Parameters

masterKeyAlias - the alias of the key to be used as a wrapping key.

failIfExists - a flag indicating whether the method should fail if a key already exists under the provided alias or return normally without overriding the key.

Exceptions

CryptoServiceBadRequestException - if a key already exists under this alias (and failIfExists is set to true].

CryptoServiceException - for general cryptographic exceptions.
