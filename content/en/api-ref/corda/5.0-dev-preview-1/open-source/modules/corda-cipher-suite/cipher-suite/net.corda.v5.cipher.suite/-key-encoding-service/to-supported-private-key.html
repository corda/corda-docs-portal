toSupportedPrivateKey

abstract fun toSupportedPrivateKey(key: PrivateKey): PrivateKey

Convert a private key to a supported implementation. This can be used to convert a SUN's EC key to an BC key.

Exceptions

IllegalArgumentException - on not supported scheme or if the given key specification is inappropriate for a supported key factory to produce a private key.
