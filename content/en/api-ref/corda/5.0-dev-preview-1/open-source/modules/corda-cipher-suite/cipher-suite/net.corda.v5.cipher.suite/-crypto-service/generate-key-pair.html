generateKeyPair

abstract fun generateKeyPair(alias: String, scheme: SignatureScheme): PublicKey

Generate and store an asymmetric key pair. Note that schemeNumberID is Corda specific. Cross-check with the network operator for supported schemeNumberID and their corresponding signature schemes. The main reason for using schemeNumberID and not algorithm OIDs is because some schemes might not be standardised and thus an official OID might for this scheme not exist yet.

Returns the PublicKey of the generated key pair.

Exceptions

CryptoServiceBadRequestException - if the scheme is not supported.

CryptoServiceException - for general cryptographic exceptions.
