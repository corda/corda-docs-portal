abstract fun sign(alias: String, data: ByteArray, scheme: SignatureScheme, signAlgorithm: String?): ByteArray

Sign a byte array using the private key identified by the input alias. Returns the signature bytes formatted according to the signature scheme. If specified the signAlgorithm determines the signature scheme used for signing.

Exceptions

CryptoServiceBadRequestException - if the private key does not exists under the alias, the key scheme is not supported, or the data is empty array..

abstract fun sign(wrappedKey: WrappedPrivateKey, data: ByteArray): ByteArray

Unwraps the provided wrapped key, using the specified wrapping key and signs the provided payload.

Parameters

wrappedKey - the private key to be used for signing in a wrapped form.

data - data to be signed.

Exceptions
