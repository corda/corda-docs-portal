generateWrappedKeyPair

abstract fun generateWrappedKeyPair(masterKeyAlias: String, wrappedKeyScheme: SignatureScheme): WrappedKeyPair

Generates an asymmetric key pair, returning the public key and the private key material wrapped using the specified wrapping key.

Parameters

masterKeyAlias - the alias of the key to be used as a wrapping key.

wrappedKeyScheme - the scheme of the key pair to be generated.

Exceptions

CryptoServiceBadRequestException - if the wrapping key does not exists under the alias or the wrappedKeyScheme is not supported.

CryptoServiceException - for general cryptographic exceptions.
