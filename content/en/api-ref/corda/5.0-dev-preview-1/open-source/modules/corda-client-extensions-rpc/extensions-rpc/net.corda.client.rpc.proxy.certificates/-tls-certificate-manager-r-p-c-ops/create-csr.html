createCsr

abstract fun createCsr(alias: String, subjectName: String): String

Generates a certificate signing request for the input subject.

Parameters

alias - The identifying alias of the key pair to be used in generating the certificate signing request.

subjectName - The subject associated with the certificate signing request. subjectName may contain the email attribute under either of the case-insensitive tokens 'EMAIL=' and 'EMAILADDRESS='.

Exceptions

CertificateOpsException - If certificate signing request cannot be created.

Return
The generated certificate signing request in PEM format. Note that if subjectName contains the email attribute, the subject of the resulting certificate signing request would hold this under the token 'E='.
