namedParameters

val namedParameters: Map<String, RpcNamedQueryParameterJson>

The named parameters to be set in the named query. The keys are the names of parameters, the RpcNamedQueryParameterJson contains JSON marshalled representation of the named parameter value.

For example, to supply named parameters with a stateStatus and a list of contractStateClassNames:

val namedParameters = mapOf(
    "stateStatus" to "\"UNCONSUMED\"",
    "contractStateClassNames" to RpcNamedQueryParameterJson("[\"com.mycordapp.MyState\", \"com.mycordapp.MyOtherState\"]")
)

To supply named parameters with a list of uuids:

val namedParameters = mapOf(
    "uuids" to RpcNamedQueryParameterJson("[\"531a6660-7cdb-4fee-b8c9-c9fea484651d\", \"751a6660-7cdb-4fee-b8c9-c9fea484651d\"]")
)

To supply named parameters with an instant timestamp:

val namedParameters = mapOf(
    "startInstant" to RpcNamedQueryParameterJson("\"2021-06-23T05:24:39.322390Z\"")
)

To supply named parameters with a custom object that could represent an EmbeddedId field:

data class MyCustomObjectPojo(val id: String, val index: Int, val name: String)
val namedParameters = mapOf("myCustomObject" to RpcNamedQueryParameterJson("{\"id\":\"8726660-7cdb-4fee-b8c9-c9fea484651d\",\"index\":1,\"name\":\"someCustomField\"}"))

To supply named parameters with a list of custom objects:

val namedParameters = mapOf(
    "myCustomObjects"
    to
    RpcNamedQueryParameterJson("[{\"id\":\"8726660-7cdb-4fee-b8c9-c9fea484651d\",\"index\":1,\"name\":\"someCustomField\"},{\"id\":\"6245660-7cdb-4fee-b8c9-c9fea484651d\",\"index\":2,\"name\":\"someCustomField2\"}]")
)
