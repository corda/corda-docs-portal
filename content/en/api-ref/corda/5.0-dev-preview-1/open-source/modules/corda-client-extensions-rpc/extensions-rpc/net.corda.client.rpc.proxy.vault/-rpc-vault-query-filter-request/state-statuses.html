stateStatuses

val stateStatuses: Set<RpcVaultStateStatus>?

The state status selection criteria for the output states. If the field is not specified, no filtering will be performed based on the state status. RpcVaultStateStatus.UNKNOWN cannot be included into this non-empty set.
