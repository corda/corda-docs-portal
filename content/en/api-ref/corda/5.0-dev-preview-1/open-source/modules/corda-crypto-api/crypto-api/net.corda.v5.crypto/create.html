fun DigestService.create(str: String): SecureHash

Converts a digest represented as a {algorithm:}hexadecimal String into a SecureHash.

This function validates the length of the created digest. Prefer the usage of this function over SecureHash.create.

Parameters

str - The algorithm name followed by a delimiter and the sequence of hexadecimal digits that represents a digest.

Exceptions

IllegalArgumentException - The input string does not contain the expected number of hexadecimal digits, or it contains incorrectly-encoded characters.
