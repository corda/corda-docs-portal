UtxoFilteredTransactionBuilder

@DoNotImplement
interface UtxoFilteredTransactionBuilder

UtxoFilteredTransactionBuilder specifies what component groups to include in a UtxoFilteredTransaction. Any component groups not specified by the builder are filtered out, so that their content or proof of existence is not included in the resulting UtxoFilteredTransaction.
