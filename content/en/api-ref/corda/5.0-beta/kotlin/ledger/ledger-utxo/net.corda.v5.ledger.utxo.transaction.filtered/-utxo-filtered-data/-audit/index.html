interface Audit<T> : UtxoFilteredData<T>

Interface for an audit proof.

This allows us to retrieve some or all of the original entries. The size entry will be the size of the component group in the original transaction. The values will be a list of all entries that we get access to - its size can be less than the size value. The values are given as a map of original index to value.
