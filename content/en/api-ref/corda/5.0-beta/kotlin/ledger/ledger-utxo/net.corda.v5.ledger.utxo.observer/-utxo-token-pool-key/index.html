UtxoTokenPoolKey

data class UtxoTokenPoolKey(val tokenType: String?, val issuerHash: SecureHash, val symbol: String)

The UtxoTokenPoolKey defines a key for a pool of similar tokens.

The full token pool key includes Holding ID, token type, issue, notary and symbol. The platform provides the holding ID, notary and optional token type.
