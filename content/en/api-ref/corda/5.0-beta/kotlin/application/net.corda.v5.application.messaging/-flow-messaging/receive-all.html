receiveAll

@Suspendable
abstract fun <R : Any> receiveAll(receiveType: Class<out R>, sessions: Set<FlowSession>): List<R>

Suspends until a message has been received for each session in the specified sessions.

Consider receiveAllMap(sessions: Map>): Map when sessions are expected to receive different types.

Parameters

receiveType

type of object to be received for all sessions.

sessions

Set of sessions to receive from.

Throws

CordaRuntimeException

if any session is closed or in a failed state.
