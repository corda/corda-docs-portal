AccountMeta

class AccountMeta(val pubkey: Pubkey, val isSigner: Boolean, val isWritable: Boolean)

Represents a Solana account which is used during instruction execution. This is equivalent to the Rust struct of the same name.

If this account is a signer (isSigner is true) then the Solana notary is required to have access to the corresponding private key and sign the commit transaction as pubkey.
