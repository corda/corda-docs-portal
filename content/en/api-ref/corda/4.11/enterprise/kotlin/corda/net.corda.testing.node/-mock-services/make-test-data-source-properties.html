@JvmStatic fun makeTestDataSourceProperties(nodeName: String = SecureHash.randomSHA256().toString(), inMemory: Boolean = false): Properties

Make properties appropriate for creating a DataSource for unit tests. Defaults configuration of on-disk H2 instance. If 'databaseProvider' system property is set then creates a config from the relevant config file is present in resources folder (used to parametrize test to run against a remote database).

Parameters

nodeName - Reflects the "instance" of the on-disk database or database username/schema. Defaults to a random string.

inMemory - Set to true to use an in-memory database rather than an on-disk database. Defaults to true.
