  • All Implemented Interfaces:

public final class ContractUpgradeFilteredTransaction
extends CoreTransaction

    A filtered version of the ContractUpgradeWireTransaction. In comparison with a regular FilteredTransaction, there is no flexibility on what parts of the transaction to reveal â€“ the inputs, notary and network parameters hash fields are always visible and the rest of the transaction is always hidden. Its only purpose is to hide transaction data when using a non-validating notary.
