    Annotation Description
    ConstructorForDeserializationAnnotation indicating a constructor to be used to reconstruct instances of a class during deserialization.
    CordaSerializationTransformEnumDefaultsThis annotation is used to mark an enumerated type as having had multiple members added, It acts as a container annotation for instances of CordaSerializationTransformEnumDefault, each of which details individual additions.
    CordaSerializationTransformEnumDefaultThis annotation is used to mark an enumerated type as having had a new constant appended to it.
    CordaSerializationTransformRenamesThis annotation is used to mark a class as having had multiple elements renamed as a container annotation for instances of CordaSerializationTransformRename, each of which details an individual rename.
    CordaSerializationTransformRenameThis annotation is used to mark a class has having had a property element.
    DeprecatedConstructorForDeserializationThis annotation is a marker to indicate which secondary constructors should be considered, and in which order, for evolving objects during their deserialization.
    CordaSerializableThis annotation is a marker to indicate that a class is permitted and intended to be serialized as part of Node messaging.
    SerializableCalculatedPropertyUsed to annotate methods which expose calculated values that we want to be serialized for use by the class carpenter.
