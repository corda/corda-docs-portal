filterReferenceInputs

fun <T : ContractState> filterReferenceInputs(clazz: Class<T>, predicate: Predicate<T>): List<T>

Helper to simplify filtering reference inputs according to a Predicate.

Return

the possibly empty list of reference states matching the predicate and clazz restrictions.

Parameters

clazz

The class type used for filtering via an Class.isInstance check. clazz must be an extension of ContractState.

predicate

A filtering function taking a state of type T and returning true if it should be included in the list. The class filtering is applied before the predicate.
