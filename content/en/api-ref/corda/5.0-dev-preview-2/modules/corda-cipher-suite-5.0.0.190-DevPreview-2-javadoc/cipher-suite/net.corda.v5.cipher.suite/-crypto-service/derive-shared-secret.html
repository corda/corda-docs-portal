deriveSharedSecret

abstract fun deriveSharedSecret(spec: SharedSecretSpec, context: Map<String, String>): ByteArray
Content copied to clipboard

Optional, derives Diffie–Hellman key agreement shared secret by using the private key associated with SharedSecretSpec.publicKey and SharedSecretSpec.otherPublicKey, note that the key schemes of the SharedSecretSpec.publicKey and SharedSecretSpec.otherPublicKey must be the same and the scheme must support the key agreement secret derivation.

Return

the shared secret. Note that it's expected that the returned secret must be not post processed by applying further transformations (e.g. HKDF or any other) as that will be done by Corda itself.

Parameters

spec

the operation parameters, see SharedSecretAliasSpec and SharedSecretWrappedSpec

context

the optional key/value operation context. The context will have at least one variable defined - 'tenantId'.

Throws

kotlin.IllegalArgumentException

if the key is not found, the key scheme is not supported, the signature spec is not supported, the key schemes of the public keys are not matching or in general the input parameters are wrong

kotlin.UnsupportedOperationException

if the operation is not supported

net.corda.v5.crypto.exceptions.CryptoException

, non-recoverable
