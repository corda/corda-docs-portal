Digest extensions: Interface defining a factory creating a custom digest implementation. The interface should be implemented if a CPK developer wishes to provide support for digest algorithms beyond supported by the Corda Platform. For each algorithm there must be matching a pair of DigestAlgorithmFactory and DigestAlgorithm implementations.

See also

net.corda.v5.crypto.extensions.DigestAlgorithm

for an example on how to use it.
