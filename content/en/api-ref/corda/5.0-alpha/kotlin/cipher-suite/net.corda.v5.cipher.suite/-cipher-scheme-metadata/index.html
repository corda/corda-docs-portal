CipherSchemeMetadata

interface CipherSchemeMetadata : KeyEncodingService, AlgorithmParameterSpecEncodingService

Service which provides metadata about cipher suite, such as available key schemes, digests, security providers, key factories, and SecureRandom instance as well as some utility methods.

Infers the signature spec from the PublicKey and DigestAlgorithmName. The digest may be ignored for some public key types as the digest is integral part of the signing/verification, e.g. if the publicKey is 'EdDSA' then the digest is set to "EdDSA" by the platform's default implementation.
