createWrappingKey

abstract fun createWrappingKey(masterKeyAlias: String, failIfExists: Boolean, context: Map<String, String>)

Optional, generates a new key to be used as a wrapping key. Some implementations may not have the notion of the wrapping key in such cases the implementation should do nothing (note that REQUIRE_WRAPPING_KEY should not be listed for such implementations).

Parameters

masterKeyAlias

the alias of the key to be used as a wrapping key.

failIfExists

a flag indicating whether the method should fail if a key already exists under the provided alias or return normally without overriding the key.

context

the optional key/value operation context.

Throws

IllegalArgumentException

if the failIfExists is set to true and the key exists

UnsupportedOperationException

if the operation is not supported

CryptoException

, non-recoverable
