fun computeHSMAlias(tenantId: String, alias: String, secret: ByteArray): String

Computes an alias based on the value supplied by the tenant, as the HSM can be shared across several tenants that provide a level of separation.

The default implementation computes HMAC (HmacSHA256) for concatenation of the tenant's id and their alias, transforms it to HEX, and takes the first 12 characters of that result.

Return

A computed alias that must be unique and deterministic. The same inputs (tenantId and alias) always produce the same output.

Parameters

tenantId

The tenant's id which the alias belongs to

alias

Alias as supplied by the tenantId.

secret

Secret.

Throws

IllegalArgumentException

if the tenant, alias or secret are empty.
