abstract fun sign(spec: SigningSpec, data: ByteArray, context: Map<String, String>): ByteArray

Signs a byte array using the private key identified by the input arguments.

Return

the signature bytes formatted according to the default signature spec.

Parameters

spec

(either SigningAliasSpec or SigningWrappedSpec) to be used for signing.

data

the data to be signed.

context

the optional key/value operation context. The context will have at least one variable defined - 'tenantId'.

Throws

IllegalArgumentException

if the key is not found, the key scheme is not supported, the signature spec is not supported or in general the input parameters are wrong

CryptoException

, non-recoverable
