@JvmStatic
fun toAttributesMap(name: String): Map<String, String>

Parses the string representation of X500 name and builds the attribute map.

The key is the attributes keys, like CN, O, etc. Constraints:

  • The RDNs cannot be multivalued

  • The attributes must have single value

  • The only supported attributes are C, ST, L, O, OU, CN

  • Attributes cannot be duplicated

Return

The attribute map parsed from the name.

Parameters

name

The string representation to build the attribute map from.

Throws

IllegalArgumentException

if required attributes are missing, constrains are not satisfied or the name is improperly specified.
