collectRPCAuditData

abstract fun collectRPCAuditData(format: AuditDataRPCOps.Format = Format.JSON, username: String? = null, action: String? = null, startTime: Instant? = null, endTime: Instant? = null): String

Collects the RPC audit data for the given parameters and returns it in the requested format.

Return

A string containing the audit data in the requested format.

Parameters

format

The output format of the audit data. Defaults to Format.JSON.

username

Optional filter to only return entries for a specific username.

action

Optional filter to only return entries matching the given action name.

startTime

Optional lower bound for the timestamp of audit entries.

endTime

Optional upper bound for the timestamp of audit entries.
