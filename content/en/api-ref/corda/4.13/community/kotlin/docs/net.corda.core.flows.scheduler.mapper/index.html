@Target(allowedTargets = [AnnotationTarget.CLASS])
annotation class FlowThreadPool(val value: String)

This annotation specifies the preferred thread pool to be used for running the flow. Note that this does not guarantee the flow will run in the specified thread pool, as the thread pool might not be configured.
