CryptoServerProviderConfig(Device: String, ConnectionTimeout: Int = 30000, Timeout: Int = 60000, KeepSessionAlive: Int = 0, KeyGroup: String = "*", KeySpecifier: Int = -1)

Configuration class for CryptoServerProvider. Currently not supported: DefaultUser,KeyStorePath,LogFile,LogLevel,LogSize.

Parameters

Device - combination of host and port of the HSM, of the form "$port@$host".

ConnectionTimeout - maximum wait in milliseconds upon connection to the device.

Timeout - maximum wait in milliseconds if the device is not responding.

KeepSessionAlive - if set to false, the session will expire after 15 minutes.

KeyGroup - the key group to be used when querying and generating keys.

KeySpecifier - the key specifier used when querying and generating keys.
