connectAndAuthenticate

fun <T> connectAndAuthenticate(headless: Boolean = false, renderer: Renderer = LoggingRenderer(ctx), logOff: Boolean = true, userInfo: <ERROR CLASS>? = null, block: (CryptoService<U, V>) -> T): T

Interactively (using console) authenticates a user against the crypto service. Once authentication is successful the block is executed.

Parameters

headless - indicate if user input is available, the authenticator will prompt for extra authentication input if it is not in headless mode.

renderer - object responsible for handling of message rendering and reading of user input.

block - to be executed once the authentication process succeeds.

logOff - flag indicating whether the crypto service should log off should after block execution.
