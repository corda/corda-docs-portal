GlobalCryptoLock

object GlobalCryptoLock

A global lock object so only one thread can connect and authenticate with a crypto service at a time. This allows the crypto service interaction to complete with logOut so the isAuthenticated method doesn't return true in a different thread before the first thread has called logOut.
